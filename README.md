# portfolio-ciber-risco-auditoria
Portfólio técnico em segurança, auditoria, IA e risco

---

PoF: Arquitetura voltada para...
- Identificar risco
- Balancear custo
- Garantir continuidade
- Garantir poder de negociação
- Garantir governança

OBS: a opção pelo Azure para estudo de caso deu-se principalmente em razão da compatibilidade contratual da interface de empresa Microsoft com diversos contratos estabelecidos com o poder público, ambiente com maior rigidez a mudanças e com maior vulnerabilidade à dependencia. As simulações poderiam ter sido feitas com outras Clouds como AWS, GCP, IBM CLoud ou outra.
