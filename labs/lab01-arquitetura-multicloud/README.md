# LAB 01 — Arquitetura Multicloud Governamental

## 1. Objetivo
Projetar uma arquitetura híbrida e multicloud
para reduzir lock-in e custos operacionais.

## 2. Problema
- Lock-in tecnológico
- Dependência contratual
- Custo imprevisível (OPEX)
- Risco de indisponibilidade concentrada

## 3. Escopo
- Nuvem Pública: Azure
- Nuvem Privada: OpenStack
- On-Prem: Data Center próprio
- Camada de Segurança Unificada
- Orquestração central
- Separar carga crítica, carga elástica e carga sensível

## 4. Metodologia
Estudo de arquitetura + simulação.

## 5. Entregáveis
- Diagrama
- Relatório técnico
- Análise de risco

## 6. Aplicações Práticas
Setor público, bancos e grandes empresas.
