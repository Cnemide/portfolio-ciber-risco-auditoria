## modelagem estratégica.

Cenário A — 100% Azure
- Não há investimento em hardware.
- Não há instalação física.
- Provisionamento rápido.
- Escalabilidade paga por uso.
- Banco gerenciado custa mais.
- Tráfego inter-região.

Cenário B — Híbrido (Azure/Pública + OpenStack/Privada + On-Prem)
- Parte vai para cloud.
- Parte exige infraestrutura local.
- Exige integração.
- Cargas previsíveis ficam on-prem.
- Cloud fica para elasticidade.

Cenário C — Multicloud
- Requer orquestração.
- Requer engenharia de integração.
- Requer equipe mais qualificada.
- Mobilidade de cargas conforme custo.
- Pode evitar dependência.
