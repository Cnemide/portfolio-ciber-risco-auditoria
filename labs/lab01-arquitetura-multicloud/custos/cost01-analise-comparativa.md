# cost01 — Simulação Comparativa de Custos

## 1. Cenários Avaliados

### Cenário A — 100% Azure
- Infraestrutura como serviço
- Banco de dados gerenciado
- Monitoramento integrado

### Cenário B — Híbrido
- Azure para aplicações públicas
- OpenStack para processamento interno
- On-prem para banco crítico

### Cenário C — Multicloud distribuído
- Azure + OpenStack + replicação cruzada
- Orquestração centralizada

---

## 2. Comparativo Conceitual

| Critério | 100% Azure | Híbrido | Multicloud |
|-----------|-------------|----------|-------------|
| Custo inicial | Baixo | Médio | Alto |
| Custo longo prazo | Alto | Médio | Controlável |
| Lock-in | Alto | Médio | Baixo |
| Complexidade | Baixa | Média | Alta |
| Resiliência | Média | Alta | Muito Alta |

---

## 3. Conclusão

A arquitetura multicloud apresenta maior custo inicial, porém reduz dependência estratégica e melhora previsibilidade de risco institucional.
