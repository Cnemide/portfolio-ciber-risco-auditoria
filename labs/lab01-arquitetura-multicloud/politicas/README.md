# Políticas de Governança

---

## Portabilidade estrutural.

Docker: containeriza aplicação.
- Sem container: aplicação depende do ambiente.
- Com container: pacote portátil.

Kubernetes: camada abstrata
- Funciona em Azure.
- Funciona em OpenStack.
- Funciona on-prem.

Terraform
- Infraestrutura como código:
- Evita dependência manual.
- Permite replicar ambiente.
- Permite reconstruir em outro provedor.
